## Image Guidelines:

- **Quality**: Use high-resolution images (minimum 800x600px)
- **Format**: JPG is recommended for photos (smaller file size)
- **Size**: Optimize images to be under 500KB each for faster loading
- **Aspect Ratio**: 4:3 ratio works best for the layout
- **Content**: Show actual products in a professional, well-lit setting
